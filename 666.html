使用VS Code Live Server打开网页

简单几步，轻松实现实时预览

1

安装Live Server扩展

打开VS Code，进入扩展市场，搜索"Live Server"并安装它。

2

打开HTML文件

在VS Code中打开你的HTML文件，确保它是当前活动标签页。

3

启动Live Server

右键单击HTML文件，选择"Open with Live Server"，或点击右下角的"Go Live"。

操作演示

VS Code // 编辑器界面
└── Extensions // 按Ctrl+Shift+X
    └── Search: "Live Server"
        └── Install // 点击安装

Right-click on HTML file → "Open with Live Server"
// 或者
Click "Go Live" in status bar

Browser opens automatically at: http://127.0.0.1:5500/

使用技巧

  • Live Server会自动检测文件更改并刷新浏览器
  • 默认端口是5500，如果被占用会自动选择其他端口
  • 可以在扩展设置中自定义默认配置
  • 支持同时开启多个Live Server实例
